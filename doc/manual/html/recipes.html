Recipes and common tasks¶

Analysing a tree¶

Problem: you want to analyse a tree for shape metrics / topology etc.

Steps:

  1. Get the tree into Nexus format. The easiest way is probably to paste a Newick tree into a suitable Nexus file.
  2. Run Mesa, load the tree.
  3. Enter the instant analysis menu
  4. Select the relevant analyses.
